## 쌍용교육센터 성적관리 시스템

* 검색 결과 데이터 요청 및 출력: jQuery, Ajax, JSON
* 웹 화면 구성: HTML, CSS, BootStrap
* 결과를 얻어오기 위한 서버 액션: JAVA, Servlet, JSP, EL
* DB액션 처리: JDBC
* 개발 Tool : Eclipse, SVN, Apache Tomcat v8.0, VisualSVN Server Manager
